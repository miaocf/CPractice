## 3. 基础使用

| 命令       | 作用               | 说明                                                         |
| ---------- | ------------------ | ------------------------------------------------------------ |
| git add .  | 添加文件           |                                                              |
| git commit | 提交               | 会弹出一个VIM 的窗口，输入 `i` 进入编辑模式， 写完后输入 `:wq` 退出编辑模式 |
| git log    | 查看提交内容       |                                                              |
| git statue | 查看工作区状态     |                                                              |
| git push   | 推送分支           | 有一个 -u 参数                                               |
| git –amend | 修改Commit message |                                                              |
